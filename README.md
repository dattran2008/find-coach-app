# **Find Coach app - Vue 3**

## Introduction
- This is a small web app which let user find their own coach to train them about areas of expertise (ex: Front-End, Back-end, Career,...)

- ### :hammer: **It is built on**:
   - [x] Frontend: Vue 3
   - [x] Backend: Firebase API 

## Project setup
```
npm install
```
